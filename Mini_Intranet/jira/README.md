# Jira – AlpineSystems AG

## Zweck
Jira dient als Ticket-System für Aufgaben, Bugs und Projekte der AlpineSystems AG.
Beispielprojekt: "AlpineCore".

## Zugriff
- URL: http://localhost:8080
- Admin-User wird beim ersten Start im Browser eingerichtet.

## Persistenz
Die Daten werden im Volume `jira_data` gespeichert:
- Pfad im Container: /var/atlassian/application-data/jira
